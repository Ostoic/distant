Detailed Description

template<typename KernelObject>
class distant::kernel_objects::snapshot< KernelObject >

A snapshot is a read-only copy of the current state of one or more of the following lists that reside in system memory: processes, threads, modules, and heaps. snapshot is a range modelling InputRange whose elements consist of valid instances of the specified KernelObject.

Template Parameters
KernelObjectmust be one of the following: process, thread, heap, module.
